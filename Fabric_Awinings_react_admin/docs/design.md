# Дизайн (описание для разработчика)

Концепция визуала и токены вёрстки для сайта «Фабрика Тентов». Подключение шрифтов, переменные Tailwind/CSS и поэкранные размеры — в **[typography.md](./typography.md)**.

---

## 1. Цель дизайна

Создать **современный**, **«воздушный»**, **продающий** дизайн, который вызывает **доверие** и **желание заказать услугу**. Дизайн должен быть **лёгким**, не перегруженным, с **плавными формами** и **акцентами на ключевых элементах**.

**Ключевые ощущения**, которые передаёт интерфейс:

| Ощущение | Смысл для пользователя |
|----------|-------------------------|
| **Надёжность** | Компанию можно доверить заказ |
| **Качество** | Продукт и сервис на уровне |
| **Открытость** | Нет скрытых условий, всё понятно |
| **Современность** | Актуальный, «живой» бренд |
| **Забота о клиенте** | Удобство и внимание к деталям |

Ниже — **§2. Глобальные стили** (цвета, шрифты и типографика, скругления, тени, отступы и сетка).

## 2. Глобальные стили

### 2.1. Цветовая схема

| Назначение | HEX | Применение |
|------------|-----|------------|
| Основной фон | `#FEFBF5` | Тёплый бежево-белый — фон **всей страницы** |
| Акцент (оранжевый) | `#E87A00` | Кнопки **Primary**, иконки, ссылки, **цены**, акцентные элементы, **hover**-состояния |
| Вторичный (зелёный) | `#2D5A27` | Дополнительные акценты, иконки преимуществ, акцентные рамки |
| Текст основной | `#1A1A1A` | Заголовки и основной текст |
| Текст второстепенный | `#666666` | Подзаголовки, описания |
| Текст светлый | `#999999` | Подписи, даты, мелкие элементы |
| Белый | `#FFFFFF` | Фон **карточек**, **модальных окон**; текст на кнопках **Secondary** |
| Границы и разделители | `#E5E5E5` или `#F0F0F0` | Линии, обводки, разделители |

Пример для CSS (`:root`):

```css
:root {
  --color-bg-base: #FEFBF5;
  --color-accent: #E87A00;
  --color-secondary: #2D5A27;
  --color-text: #1A1A1A;
  --color-text-muted: #666666;
  --color-text-subtle: #999999;
  --color-surface: #ffffff;
  --color-border: #e5e5e5;
  --color-border-light: #f0f0f0;
}
```

### 2.3. Скругления

| Элемент | `border-radius` |
|---------|-----------------|
| Мелкие элементы (иконки, аватарки) | **8px** |
| Кнопки | **40px** (полностью скруглённые, «пилюли») |
| Карточки товаров и отзывов | **16px** |
| Крупные блоки (Hero, калькулятор, форма) | **24px** |
| Модальные окна | **24px** |

### 2.4. Тени

**Карточка (стандарт):** смещение X **0**, Y **12px**, размытие **24px**, растяжение **-8px**, цвет `rgba(0, 0, 0, 0.08)`.

```css
box-shadow: 0 12px 24px -8px rgba(0, 0, 0, 0.08);
```

**Карточка при hover:** Y **16px**, размытие **32px**, растяжение **-12px**, цвет `rgba(0, 0, 0, 0.12)`.

```css
box-shadow: 0 16px 32px -12px rgba(0, 0, 0, 0.12);
```

**Кнопка Primary:** Y **4px**, размытие **8px**, растяжение **0**, цвет `rgba(232, 122, 0, 0.25)`.

```css
box-shadow: 0 4px 8px 0 rgba(232, 122, 0, 0.25);
```

### 2.5. Отступы и сетка

**Десктоп** (ориентир ширины артборда **1920px**):

| Параметр | Значение |
|----------|----------|
| Контейнер контента | **1280px**, по центру (`margin-left/right: auto`) |
| Горизонтальные отступы от краёв **контейнера** | **24px** (внутренние поля контента внутри блока 1280px) |
| Вертикальные отступы **между секциями** | **96px** |
| Внутренние отступы секции сверху/снизу | **0** — ритм секций задаётся **внешними** отступами между секциями |

**Мобильный** (ориентир **390px**):

| Параметр | Значение |
|----------|----------|
| Горизонтальные отступы от краёв экрана | **16px** |
| Вертикальные отступы между секциями | **48px** |
| Контейнер 1280px | **не используется** — контент на всю ширину с боковыми полями |

**Сетка (десктоп):** **12 колонок**, ширина колонки вычисляется автоматически, **gutter** между колонками **24px**.

Глобальные **компоненты** (кнопки, input, карточки, иконки МП, бургер) — **[components.md](./components.md)**.

**Анимации** (макет для дизайнера, реализация Framer Motion) — **[animations.md](./animations.md)**.

## 7. Дополнительные требования к дизайну

- Все **изображения и фото** — **реальные** съёмки или **качественный сток** (заказчик может заменить контент позже).
- Дизайн **чистый**, без лишних декоративных элементов.
- **Акцентные** элементы (кнопки, цены, ссылки) — **только оранжевым** акцентом (`#E87A00` и варианты из палитры §2.1).
- **Зелёный** (`#2D5A27`) — **дозированно**: иконки преимуществ, вторичные акценты (см. §2.1).
- Заголовки **Playfair Display** — достаточный **межбуквенный интервал** (**tracking** примерно **-0.01…-0.02em**, согласовано с [§ 2.2](./design.md#22-шрифты-и-типографика)).
- Текст **Inter** — комфортная **высота строки `line-height` 1.4–1.5**.

## Проверка перед сдачей (чек-лист)

Использовать при приёмке **макета** (Figma) и финальной **вёрстки**.

- [ ] Все **цвета** соответствуют цветовой схеме (§2.1).
- [ ] Все **шрифты** соответствуют типографике (§2.2, [typography.md](./typography.md)).
- [ ] У всех **кнопок** есть состояния **normal / hover / active** (и **disabled** для Primary — [components.md](./components.md)).
- [ ] У всех **полей ввода** — состояния **normal / focus / error**.
- [ ] **Адаптив:** на мобильной версии **нет горизонтального скролла**.
- [ ] **Адаптив:** на мобильной версии **кнопки** комфортного размета (**от 44px** по высоте или касаемой области — см. минимум тач-таргета).
- [ ] **Контраст** текста соответствует **WCAG**: **4.5:1** для основного текста.
- [ ] **Отступы между секциями** **одинаковы** по всему макету (§2.5).
- [ ] **Карточки в сетках** выровнены **по высоте**.
