Микро-иконки маркетплейсов для шапки (компактные пилюли)
============================================================

РАЗМЕР И ХОЛСТ
  • Логический размер в вёрстке: ~14–16 CSS px внутри круга 24×24 px.
  • Исходник: квадрат 24×24 px (viewBox для SVG) — базовый вариант.
  • Retina: экспорт дубликата 48×48 px (или один SVG — он масштабируется без потери чёткости).

ФОРМАТ (по приоритету)
  1) SVG — один файл, лёгкий, чёткий на любом DPR. Упростить пути, убрать лишние группы/метаданные
     из Illustrator, по возможности оптимизировать (SVGO).
  2) WebP с альфа-каналом — если дают только растр; качество ~90, без артефактов по краям.
  3) PNG 32-бит с прозрачностью — универсально.

СОДЕРЖИМОЕ ЗНАЧКА
  • Только знак/иконка бренда, без полного горизонтального логотипа с текстом.
  • Внутри квадрата оставьте «воздух» 2–3 px по краю (safe area), чтобы в круглой маске не
    резало усы/буквы.
  • На тёмном и светлом фоне читаемость: при необходимости лёгкая обводка или контрастный фон
    внутри самого SVG (как у плейсхолдеров wb-24.svg / ozon-24.svg).

ЮРИДИЧЕСКИ
  • Используйте материалы из партнёрских/пресс-китов Wildberries и Ozon; плейсхолдеры в репозитории
    — временные, замените на официальные файлы перед продакшеном.

ПОДКЛЮЧЕНИЕ В ПРОЕКТЕ
  • Пути задаются в frontend/src/config/site.ts (поле iconSrc у записи маркетплейса).
  • Компактный режим в шапке берёт iconSrc, иначе — полноразмерный logoSrc.
  • Текущие файлы (источник logo-teka.com, хранятся локально): wildberries-sign-logo.svg, ozon-icon-logo.svg.
