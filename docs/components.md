# Глобальные компоненты (Components)

Спецификация переиспользуемых UI-компонентов для вёрстки. Базовые токены (цвета, тени, скругления) — **[design.md](./design.md)**; шрифты — **[typography.md](./typography.md)**.

## 3.1. Кнопка Primary

| Параметр | Значение |
|----------|----------|
| Высота | **56px** |
| Горизонтальный padding | **32px** (left / right) |
| Скругление | **40px** |
| Фон | Градиент **#E87A00 → #D06B00** или сплошной **`#E87A00`** |

| Состояние | Фон | Прочее |
|-----------|-----|--------|

| **Active** | `#A84F00` | **`scale(0.98)`** |
| **Disabled** | `#CCCCCC` | Без тени; текст **`#999999`** |

## 3.5. Карточка товара

| Параметр | Значение |
|----------|----------|
| Ширина | **320px** (десктоп) / **100%** (мобильный) |
| Скругление карточки | **16px** |
| Фон | **`#FFFFFF`** |
| Тень | Стандартная карточка — [design.md § 2.4](./design.md#24-тени) |
| Внутренний padding | **16px** |

**Структура (сверху вниз)**

1. **Изображение:** **288×200px**, скругление **12px**, `object-fit: cover`.
2. Отступ **12px**.
3. **Название:** стиль **H4** (Playfair Display, *курсив*, **20px**) **или** обычный текст **18px** (уточнение по макету).
4. Отступ **8px**.
5. **Цена:** Inter **700**, **24px**, цвет **`#E87A00`**.
6. Отступ **16px**.
7. Кнопка **«Заказать»** — **Primary**, **на всю ширину** карточки (внутри padding).

**Hover карточки:** тень **усиленная** ([hover-тень](./design.md#24-тени)); лёгкий подъём **`translateY(-4px)`**.

## 3.7. Карточка отзыва

| Параметр | Значение |
|----------|----------|
| Ширина | **400px** (десктоп) / **100%** (мобильный) |
| Скругление | **16px** |
| Фон | **`#F5F0E8`** или **белый** (по макету / секции) |
| Внутренний padding | **24px** |

**Структура**

- **Верх:** фото клиента **80×80px**, **круг** (`border-radius: 50%`), `object-fit: cover`; **имя** — Inter **600**, **18px**.
- **Рейтинг:** 5 звёзд, иконки **16×16px**, цвет **золотой** (например палитра бренда или `#D4A574` / уточнить по макету).
- **Текст отзыва:** Inter **400**, **16px**, цвет **`#666666`**, *курсив*.
- **Опционально:** превью видео с **иконкой play** поверх изображения.

## 3.9. Бургер-меню (мобильная версия)

**Иконка (закрыто):** три горизонтальные полоски — высота каждой **2px**, ширина **24px**, цвет **`#1A1A1A`**.

**Открыто:** полоски превращаются в **крест** — верхняя и нижняя поворот **±45°**, средняя **скрыта** (`opacity: 0` или `scale(0)`).

**Выезжающая панель:** с **правой** стороны; ширина **100%** вьюпорта **или** **280px** (по макету); фон **белый**; **тень слева** у панели (визуальное отделение от контента).
