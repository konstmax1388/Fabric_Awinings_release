# Этапы разработки

Порядок работ по проекту «Фабрика Тентов». Технологии по этапам: **Vite + React**, **Tailwind CSS**, **Framer Motion**, **Django + Django REST Framework**, **JWT**, админка — **React Admin** или **кастомная панель**; SEO — **React Helmet**, sitemap, robots, микроразметка.

Детальный функционал и дизайн — [functional-requirements.md](./functional-requirements.md), [design.md](./design.md), [typography.md](./typography.md).

---

| Этап | Название | Содержание |
|------|----------|------------|
| **1** | Настройка проекта | Инициализация **Vite + React**, настройка **Tailwind**, подключение шрифтов, настройка **Django + DRF**, настройка маршрутизации (frontend + API). |
| **2** | Вёрстка главной страницы | Все **11 блоков** главной с адаптивом. |
| **3** | Анимации | Реализация анимаций из **согласованного списка** на **Framer Motion** (при появлении полного перечня — дополнить этот документ или приложение к ТЗ). |
| **4** | Страницы каталога и карточка товара | Сетка товаров, фильтры, сортировка, пагинация, фотогалерея, блок маркетплейсов. |
| **5** | Интеграция маркетплейсов | Общие ссылки в **футере** и индивидуальные ссылки в **карточках товаров** (связка с данными из backend/админки). |
| **6** | Калькулятор цены | Интерактивный калькулятор с формулой расчёта и анимацией изменения цены. |
| **7** | Backend (Django REST) | Модели: товары, портфолио, отзывы, статьи, заявки; **API-эндпоинты**; **JWT**-аутентификация. |
| **8** | Админка | React Admin **или** кастомная панель с полным **CRUD** для всего контента. |
| **9** | SEO и мета-теги | **React Helmet**, `sitemap.xml`, `robots.txt`, микроразметка. |
| **10** | Тестирование и отладка | Кроссбраузерное и мобильное тестирование, проверка производительности, исправление багов. |

---

## Замечания по плану

- Этапы **4** и **5** на практике частично пересекаются (ссылки МП в карточке); этап 5 выделен как отдельная проверка сквозной интеграции данных.
- Этап **7** по объёму может включать модели и API для **всего** редактируемого контента (в т.ч. блоки главной из §2.2.1 ТЗ); при детализации — расширить формулировку этапа 7 отдельным подпунктом.

## Статус реализации (код)

- **Этап 1** — сделано: Vite/React/Tailwind, **Django + DRF** (`backend/`), **CORS**, `docker compose` (`api` + `frontend`), **React Router**, `VITE_API_URL`, `GET /api/health/`.
- **Этап 2** — главная: **11 блоков** в `frontend/src` (`SiteHeader` … `SiteFooter`), страницы-заглушки `/catalog`, `/portfolio`, `/contacts`, `/blog`.
- **Этап 3** — **Framer Motion** на главной: пресеты в `frontend/src/lib/motion-presets.ts`, `AnimatedCounter` / `PulsingCTA`, fade-up + stagger по секциям, hover карточек, слайд отзывов, анимация смены цены в калькуляторе, бургер и мобильное меню в `SiteHeader` (см. [BACKLOG.md](../BACKLOG.md)).
- **Этап 4+** — каталог и карточка товара — в плане (см. BACKLOG).
